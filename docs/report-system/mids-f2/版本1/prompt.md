## 一、报告的核心逻辑与结构要求

### 1.1 报告的核心主线（三层递进结构）

MIDS-F2 将二代在传承情境中的创新领导力拆解为三层递进结构。整篇报告必须围绕这一主线展开，让读者理解五个维度不是并列的，而是一个有机的能力金字塔：

- **顶层·方向感**：战略破局力。解决"往哪走"的问题——能否跳出父辈的认知框架，为企业的未来找到新的增长曲线。
- **中层·落地感**：执行颠覆力 + 资源整合力。解决"怎么走"的问题——能否将想法转化为行动，并撬动必要的资源支撑变革。
- **底层·根基感**：逆商与灰度 + 伦理与格局。解决"走不走得远"的问题——能否在阻力中坚持，并在价值观上为企业的长期发展把舵。

三层之间的关系是：**没有战略方向，执行就是蛮干；没有执行能力，战略就是空想；没有底层心力与格局，走得越快，翻车风险越高。**

### 1.2 各维度水平定位标准

- 4.5–5.0：卓越
- 3.5–4.4：良好
- 2.5–3.4：基础
- 1.0–2.4：待开发

## 二、措辞与风格要求

### 2.1 核心原则

1. **数据 → 特质 → 行为 → 价值**：每一段分析必须遵循这条推导链。先给出得分，再解释这个得分代表什么特质，然后说明这种特质在工作中会表现出什么行为，最后落到对传承情境的价值或影响。

2. **结论先行**：每个维度的分析，第一句话就要给出核心判断，然后再展开论证。

3. **措辞要有锐度，但不刻薄**：不要用"这表明…这意味着…"这种模板化句式，每段换着开头。不要回避问题，但要用"发展空间"而非"缺陷"的视角来表达。

4. **语言要"说人话"**：避免堆砌学术术语，用管理者能听懂的语言表达专业判断。

5. **框架串珠**：每个维度分析完毕后，都要用 `impactOnSuccession` 字段把该维度拉回到三层结构的主线上，告诉读者这一维度的表现对整体传承意味着什么。五个维度不是孤立的分析报告，而是围绕"这个能力结构能否支撑领军接班"这一个核心问题展开的层层论证。

### 2.2 各维度分析要点

**strategic_breakthrough（战略破局力）**：
- 高分表现：逆周期布局能力、宏观趋势转化能力
- 中等表现：跨界意愿
- 解读方向：在战略冒进与稳健改良之间的平衡，审慎在制造业传承中的价值
- 影响落点：为领军接班提供了认知基础，但"看到"不等于"做到"

**execution_disruption（执行颠覆力）**：
- 高分表现：工具应用能力、试错意识
- 中等表现：数据决策习惯
- 解读方向：技术亲和力的价值、经验依赖在数字化转型中的制约
- 影响落点：不仅"看得到"也"做得了"，但需要从"工具使用者"进化为"数据驱动者"

**resource_integration（资源整合力）**：
- 高分表现：跨文化嫁接能力、老臣共情能力
- 中等表现：资本运作、职业经理人引入
- 解读方向：软性整合与硬性运作的差距、结构性资源运作经验的价值
- 影响落点：擅长内部挖潜但外部突破偏于保守

**adversity_quotient（逆商与灰度）**：
- 高分表现：长期主义思维
- 中等表现：被否决后的迂回、亲情与制度的平衡、孤独感耐受、放低身段
- 解读方向：心理韧性的真实状态、为何这是"最难通过知识学习来提升的维度"
- 影响落点：有方向有能力但扛不住压力，是年轻改革者黯然离场的根本原因。**必须在"允许犯错、允许被拒绝"的环境中积累灰度决策经验**

**ethics_vision（伦理与格局）**：
- 表现：四个条目均衡稳健
- 解读方向：现代企业家精神的系统认知、与父辈理念的张力如何转化为改革武器
- 影响落点：为领军接班提供信任基础，是改革的"理念杠杆"

### 2.3 发展建议的措辞要求

- 不要给出"具体动作"（如"三个月内做什么"），只指出**提升方向和为什么需要提升**
- 每一条建议要说清楚两件事：**在哪个维度上提升** + **为什么这个维度当前的表现会制约传承成功**
- 举例："建议深化数据驱动的决策习惯。当前在关键决策场景中仍存在经验依赖倾向，这一倾向在数字化转型中会制约决策质量与团队信任。"
