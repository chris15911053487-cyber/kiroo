## 组件规范

### 得分胶囊 (score-block)

4 个胶囊水平排列：综合胜任力指数、领导风格、人格特质、创造力。
- 圆角 `24px`（胶囊形）
- 大数字 `1.9rem` + 小标签 `0.7rem`

### 进度条 (progress-bar)

- 背景 `#e2e8f0`，填充 `#2c5f8a`
- 高度 10px，圆角 12px
- 宽度 = 总分百分比
- 出现在综合得分概览和综合评分汇总两处

### 场景匹配表 (scene-matching)

5 行：新团队组建、危机处理、日常运营、团队激励、战略转型。
4 列：典型工作场景、自然应对风格、匹配度、需要注意什么。
匹配度根据分数动态判定（✅完美匹配 / ✅基本够用 / ⚠️可发展 / ⚠️储备不足）。

### 交叉验证表 (cross-validation)

3 行：S2教练型↔管理潜能、S4授权型↔心理健康、S1指令型↔创造力潜质。
4 列：领导风格维度、对应人格特质、得分一致性、解读。
