# 16PF 人格测验（精选版）— 问卷规则

## 基本信息

| 项目 | 内容 |
|------|------|
| 问卷ID | `lzu-personality` |
| 理论来源 | 卡特尔 16PF 人格理论（精选 3 个核心维度） |
| 题目数量 | 15 题 |
| 计分方式 | 累加式（additive） |
| 维度数量 | 3 个 |
| 每题满分 | 2 分 |

## 指导语

> 本测验基于卡特尔16PF理论，选取与职业发展相关的3个核心维度（创造力潜质、心理健康水平、管理潜能），共15道题目。请根据您的实际情况选择最符合的选项。本测验为兰州大学管理学院研究生职业发展测评定制。

## 标准分转换规则

### 原始分 → 标准分（1-10）

```
ratio = 原始分 / 10（满分）
ratio ≤ 0.2 → 标准分 1-2
ratio 0.2-0.4 → 标准分 3-4
ratio 0.4-0.6 → 标准分 5-6（中等线）
ratio 0.6-0.8 → 标准分 7-8（良好线）
ratio > 0.8 → 标准分 9-10（优秀线）
```

### 标准分 → 等级

| 标准分 | 等级 | AI解读语气 |
|--------|------|------------|
| 9-10 | 优秀 | 肯定鼓励，强调这是核心优势 |
| 7-8 | 良好 | 肯定鼓励，指出可进一步提升 |
| 5-6 | 中等 | 指出发展空间，给出可操作建议 |
| 3-4 | 稍低 | 建设性建议，制定专项提升计划 |
| 1-2 | 较低 | 重点发展领域，需系统性训练 |

## 综合得分中的权重

人格特质部分在综合报告中的权重为 **40%**，满分 **40 分**。

### 计算公式

```
创造力潜质归一化 = (创造力原始分 / 10) × 100
心理健康归一化   = (心理健康原始分 / 10) × 100
管理潜能归一化   = (管理潜能原始分 / 10) × 100

人格综合分 = (创造力归一化 × 0.40 + 心理归一化 × 0.30 + 管理归一化 × 0.30) × 0.40
```

内部权重：创造力潜质 40%，心理健康 30%，管理潜能 30%。

---

## 评分解读规则（AI参考）

- **三特质组合效应**：三种特质存在互补关系。高创造力+低心理=创意多但易焦虑拖延；高管理+低心理=能带队但易过度控制；高心理+低创造=稳定但创新不足
- **等级越高越鼓励**，等级越低越建设性
- **每条分析必须引用具体分数**，不能泛泛而谈
- **"✔"开头的建议**必须具体可操作
